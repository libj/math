mul

Multiply T by T.

BigDecimal outperforms Decimal in runtime performance due to the fact that BigInteger.multiplyToLen(...) is implemented as an intrinsic, which proves to beat BigInt's critical native implementation of the same algorithm.

Decimal outperforms long in runtime performance in lieu of the need to encode and decode the decimal value from a long-encoded number.

Decimal outperform BigDecimal in heap allocation by requiring no more than the single Decimal instance and zero int instances for all operations.

long outperform Decimal and BigDecimal in heap allocation by avoiding heap allocation entirely for all operations.
